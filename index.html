- Bootcamp Santander 2024 - Backend com Java (DIO)

O Bootcamp Santander 2024 – Backend com Java, promovido em parceria com a DIO, proporcionou uma imersão completa no desenvolvimento backend utilizando a linguagem de programação Java. O programa abordou desde os fundamentos do Java até conceitos avançados, incluindo o uso de frameworks populares, práticas de boas implementações e padrões de design. A parte prática foi uma componente crucial do aprendizado, com momentos dedicados à criação de APIs RESTful, integração com bancos de dados, e construção de aplicações robustas e escaláveis.

- Trilha de Conhecimento – Banco de Dados (Fundação Bradesco)

O curso abordou a importância e a utilidade dos bancos de dados como coleções organizadas de informações estruturadas, armazenadas eletronicamente em sistemas de computadores. Pois, com a crescente valorização da informação no mundo moderno, compreendemos seu papel fundamental na tomada de decisões por pessoas e empresas. A trilha de aprendizado destacou a relevância da modelagem de dados, especialmente considerando os avanços tecnológicos dos últimos anos. Foram introduzidas terminologias essenciais para a implementação e administração de bancos de dados, além de práticas para gerenciar eficientemente as atividades relacionadas a essa função.

- Trilha de Conhecimento – Linguagem de Programação Python (Fundação Bradesco)

A linguagem de programação Python tem se destacado nos últimos anos por sua simplicidade e grande compatibilidade, funcionando eficientemente em uma ampla variedade de sistemas operacionais. Esta trilha de aprendizado começa de maneira didática, introduzindo o conceito de raciocínio lógico e demonstrando como a lógica está presente em nosso cotidiano. A trilha avança para apresentar a programação orientada a objetos (POO), detalhando seus quatro pilares fundamentais: abstração, encapsulamento, herança e polimorfismo. Além disso, são apresentados exemplos de outras linguagens orientadas a objetos, como Java, C++ e C#. O objetivo principal da trilha é combinar teoria e prática do raciocínio lógico utilizando a linguagem Python, através do desenvolvimento de programas básicos em modo console. A trilha também explora de forma simples as principais plataformas de desenvolvimento, capacitando os participantes a criar soluções eficazes.

- Curso Análise de dados como suporte à tomada de decisão (ENAP)

O curso online de Análise de Dados como Suporte à Tomada de Decisão, oferecido pela Escola Nacional de Administração Pública (ENAP), apresentou uma abordagem prática e teórica sobre como os dados podem ser utilizados para embasar decisões estratégicas. O conteúdo foi distribuído em quatro módulos principais: O Módulo 1 introduziu os conceitos fundamentais da análise de dados, fornecendo uma base sólida sobre a importância dos dados no processo decisório. O Módulo 2 abordou o pré- processamento de dados, ensinando técnicas de limpeza, transformação e organização de grandes volumes de informações. No Módulo 3, foi explorada a análise descritiva, enfatizando a importância de métricas e indicadores para compreender os dados. O curso se encerrou com o Módulo 4, que apresentou o desenvolvimento de painéis de Business Intelligence (BI), capacitando os participantes a criar visualizações interativas e relatórios dinâmicos que facilitam a interpretação dos dados.

- Curso Liderança e Gestão De Equipes(ENAP)

O curso Liderança e Gestão de Equipes, também oferecido pela ENAP, proporcionou uma visão abrangente e prática sobre como liderar e gerenciar equipes de forma eficaz. O conteúdo foi dividido em três módulos principais: O Módulo 1 abordou a essência da liderança, destacando as principais características de um líder eficaz e as diferentes abordagens para influenciar e inspirar equipes. O Módulo 2 tratou de técnicas e abordagens para o trabalho em equipe, ensinando estratégias para promover a colaboração, resolver conflitos e melhorar a comunicação dentro dos grupos. Por fim, o Módulo 3 explorou o impacto do clima organizacional no desempenho das equipes, enfatizando como um ambiente de trabalho positivo pode ser um propulsor para a produtividade e a inovação.

- Curso HTML5: Fundamentos para Construção de Páginas Web Modernas(IFRS)

O curso de HTML5, oferecido pelo Instituto Federal do Rio Grande do Sul (IFRS), apresentou os princípios essenciais para o desenvolvimento de sites modernos utilizando HTML5. O conteúdo programático iniciou com uma introdução à história, evolução e principais características do HTML5, destacando suas vantagens em relação às versões anteriores. A estruturação de páginas web com o uso de linguagens de marcação foi abordada em detalhe, junto com a inserção de imagens, links, listas e tabelas, elementos cruciais para criar páginas bem estruturadas. O curso também ensinou como trabalhar com formulários, oferecendo uma base sólida para o desenvolvimento de interfaces interativas e funcionais.
